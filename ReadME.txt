********************* Problème de connexion dans l'émulateur *********************

Sur Mac OS :

	- Ouvrir "Préférences Système"
	- Aller dans "Réseau"
	- Sélectionner le réseau qui est connecté à votre Mac et cliquer sur "Avancé"
	- Sélectionner "DNS", puis sur le bouton "+"
	- Ajouter "8.8.8.8" (DNS de Google) ou Si vous préféré OpenDNS "208.67.222.222"
	- Sélectionner "Ok" et "Appliquer"


Sur Windows & Linux:

	https://developers.google.com/speed/public-dns/docs/using
